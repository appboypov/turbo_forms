# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-06

### Added
- Form configuration and validation system for Flutter
- `TurboFormFieldConfig` for field configuration
- Integration with `turbo_notifiers` for reactive form state
- Logging integration with `turbolytics`
- UI components using `shadcn_ui`
- Email validation support
- Form field state management
- Validation utilities
